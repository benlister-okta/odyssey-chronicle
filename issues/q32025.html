Odyssey spotlight

Introducing new navigation!

Now available: unified top, side, and footer navigation components and a new app switcher to allow admins to easily move between their Okta apps. Take a look

Recent and upcoming component releases

We’ve expanded tag color options, introduced a new page header component, and added read-only states to form elements to provide you with more customization and flexibility while ensuring UI consistency and accessibility.

Tag color options

Added four additional non-semantic colors that allow tags to have more emphasis, variety, and contrast.

Page header

Allows the header to be easily customized in the page template. (Available in Figma now; code available soon.)

Read-only form elements

A read-only state for a consistent, informative experience that allows users to copy-paste, select, and tab.

Odyssey Figma release changelog

Track Figma releases in Confluence. The changelog includes updates, additions, and modifications to components and styles to help teams stay informed.

Product glossary update

Updated in a searchable spreadsheet, we’ve defined 120 previously undefined terms and added 130 new terms based on Okta's Localization Terminology.

Gen4ward THI

The Gen4ward THI enables all new development in Gen2 (Courage) apps to be built in Gen4 (Odyssey) without rewriting any existing functionality.

Component migration Figma plugin

Simplify your migration with our new Figma plugin. Designers and engineers can quickly and confidently update their designs with component and style find-and-replace, auditing, and linting.
