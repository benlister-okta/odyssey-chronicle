Recent component releases

Data view component

Presents data in a customizable format. Designers can choose the best presentation for the data or allow users to choose from a data table, a list, or a grid.

Date picker component

An input that allows users to select a past, present, or future date using a text field or by choosing the date from a calendar with timezone and min/max date support.

Layout component

A foundational component that standardizes how the body of Admin Console pages is composed.

New and notable

Designing with Okta AI

Users need to know when they are interacting with AI. Learn about the principles and elements to use when designing AI products to ensure clarity and consistency.

Accessibility and Odyssey

By using Odyssey, teams can create features and products that meet AA WCAG 2.2 accessibility guidelines.

Autopilot: the Okta UI Platform

Watch UI Guild and SOTU for announcements and demos of our new streamlined development and delivery UI Platform. Engineers on UI Guild rotations can start using it in Q3 and adoption is planned Q4.

Odyssey release process updates

Odyssey releases are now every other Wednesday. A two-day preview starts on the Monday before, giving engineers time for migration testing and access to the new version.
